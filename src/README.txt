Step counter for WoW.

Install this addon to start tracking your 'steps'.

Commands:

/steps help     - Shows help display
/steps show     - Toggles UI Display.  Steps: ### (Average : Max )
/steps lock     - Toggles UI lock.  Unlock to move it.
/steps reset    - Resets UI back to starting place
/steps chat     - Toggles the ability to share your steps by typing "{steps}" in any chat channel.


Shows the current daily steps of your character.
